# ▶️ How to Run

## Clone Repository

```bash
git clone https://github.com/maheshsankhat/Internship-Projects.git
```

## Open Project Folder

```bash
cd Internship-Projects
```

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Run Flask Project

```bash
python app.py
```

or

```bash
flask run
```

Open your browser:

```
http://127.0.0.1:5000
```
